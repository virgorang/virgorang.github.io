指针

发表于2022-03-01|更新于2022-03-01
|阅读量:

cpu 地址线：对内存哪个单元处理 内存
控制线：读还是写
数据线：数据的流向

地址：内存单元的编号（不可重复）

指针：
指针就是地址，地址就是指针
指针变量是存放内存单元地址的变量
指针的本质是一个操作寿险的非负整数

int *p  //p是变量名（本身也占一个地址） ，int *p表示该p变量只能存储int类型变量的地址

函数：
    void f(int *p） //定义了一个形参，存储int *
    {
        *p=100;
    }

    int main()
    {
        int i= 9;
        f(&i)
        printf("i=%d\n",i);
        return 0;
    }

    int a[5]=[1,2,3,4,5]
    a 指向的是数组的第一个元素的地址a[0]
    a + 1 指向的是数组的第二个元素的地址a[1]
    a[i] = *(a+i)
    %p 输出地址

指针的作用

在函数中改变主调函数中的存储单元如果通过函数值返回结果，显然结果只能有一个。

通过间接操作，在函数中改变主调函数中的存储单元，可以达到让函数带回多个计算结果到主调函数的目的。
访问动态分配的存储单元

各种指针

int *p;
整型指针，指向整数的指针
整型指针指针，指向(整型)
int **p; 指针的指针，二级(整型)指针
int a[ 10]; 整型数组
int *p[ 10]; 指针数组
int (*p)[10]; 数组指针，指向数组的指针，行指针

intf(int x); 返回整数的函数

int *f(int x); 返回整型指针的函数
int (*p)(int x); 函数指针，指向函数的指针

++和指针的结合

如(p)++. p所指向的单元的自增运算
p++， ++ 单目 右结合性，同级.又”++”为后缀先引用p当前所指地与结分.后再将p+1.此时指针指向下一个单元
++ (*p) ++前缀. 先增值后引用
前缀.先增值后引用
++ *p
